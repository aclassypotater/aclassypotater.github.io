How it works

After stdin is recieved, every string is made the same size with added spaces, this is so we can compare strings without worrying about index errors ex: string1[0] == string2[0] checks if the first letter of the strings is the same. after that we first iterate over the array only accessing the first character of every string, there is a dictionary that converts letters to numbers ex: a == 0, c == 2. everytime a letter is a higher value than the one ahead of it they get swapped in the array, this is repeated until all first characters are sorted. Next up is the hard part, we still compare the values of the letters in the second position but we only swap if the proceeding letters in both words are the same ex: swap bro and bat, but not bro and cat even though r > a on both of them. Then do that until the second characters are sorted and repeat until youve reached the final character spot (last letter on the longest word). Finally we can write to stdout.
